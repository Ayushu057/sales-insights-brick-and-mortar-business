# Sales Insights - Brick and Mortar Business

## 📊 Overview

This project focuses on analyzing sales data from a brick-and-mortar business to uncover trends, track performance, and support data-driven decision making.

## 🛠 Tools & Technologies

* SQL (Data Analysis)
* Power BI (Data Visualization)
* Python (Data Cleaning & Processing)

## 📈 Key Analysis Performed

* Sales trend analysis over time
* Region-wise and category-wise performance
* Customer segmentation insights
* KPI tracking (Revenue, Profit, Growth)

## 📊 Dashboard Features

* Interactive sales dashboard
* Monthly and yearly trends
* Top-performing customers and products
* Profitability analysis

## 🚀 Business Impact

* Identified top-performing regions
* Improved decision-making through KPIs
* Highlighted underperforming areas

## 📌 Conclusion

This project demonstrates how data analytics can drive business decisions using real-world sales data.
